# Badlands

A striking biome of red sand and terracotta cliffs layered with many colors. Gold ore generates at higher elevations than elsewhere.

## Overview

| Field         | Value                       |
|---------------|-----------------------------|
| Dimension     | Overworld        |
| Category      | Dry         |
| Temperature   | 2.0      |
| Precipitation | None (dry)               |

## Blocks & Resources

**Surface / Terrain Blocks:** Red Sand, Terracotta (all colors), Hardened Clay, Stone

**Notable Resources:** Terracotta (all colors), Red Sand, Gold Ore (elevated), Dead Bush
