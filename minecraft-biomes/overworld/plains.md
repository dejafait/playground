# Plains

A flat, open biome covered in grass with occasional oak trees and tall grass. One of the most common biomes and a frequent spawn location for new players.

## Overview

| Field         | Value                       |
|---------------|-----------------------------|
| Dimension     | Overworld        |
| Category      | Temperate         |
| Temperature   | 0.8      |
| Precipitation | Rain               |

## Blocks & Resources

**Surface / Terrain Blocks:** Grass Block, Dirt, Stone

**Notable Resources:** Oak Wood, Flowers, Seeds (from grass)

## Mobs

| Type     | Mobs                        |
|----------|-----------------------------|
| Passive  | Cow, Sheep, Pig, Chicken, Horse, Donkey, Rabbit                   |
| Neutral  | Wolf (rare)                   |
| Hostile  | Zombie, Skeleton, Creeper, Spider, Witch, Pillager                   |

## Structures

Village, Pillager Outpost, Ruined Portal
