# Forest

A dense biome filled with oak and birch trees. Wolves and mushrooms can be found here, and it's an excellent early-game source of wood.

## Overview

| Field         | Value                       |
|---------------|-----------------------------|
| Dimension     | Overworld        |
| Category      | Temperate         |
| Temperature   | 0.7      |
| Precipitation | Rain               |

## Blocks & Resources

**Surface / Terrain Blocks:** Grass Block, Dirt, Stone

**Notable Resources:** Oak Wood, Birch Wood, Mushrooms, Apples
