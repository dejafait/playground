# Cherry Grove

Introduced in 1.20, this biome features stunning pink cherry blossom trees with petal particle effects. Exclusive source of cherry wood.

## Overview

| Field         | Value                       |
|---------------|-----------------------------|
| Dimension     | Overworld        |
| Category      | Mountain         |
| Temperature   | 0.5      |
| Precipitation | Rain               |

## Blocks & Resources

**Surface / Terrain Blocks:** Grass Block, Dirt, Stone

**Notable Resources:** Cherry Wood, Cherry Petals, Pink Petals (ground)
