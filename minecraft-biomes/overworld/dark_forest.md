# Dark Forest

A dense, foreboding forest of dark oak and mushroom trees whose canopy blocks most sunlight. Hostile mobs can spawn during the day here.

## Overview

| Field         | Value                       |
|---------------|-----------------------------|
| Dimension     | Overworld        |
| Category      | Temperate         |
| Temperature   | 0.7      |
| Precipitation | Rain               |

## Blocks & Resources

**Surface / Terrain Blocks:** Grass Block, Dirt, Mycelium (patches), Stone

**Notable Resources:** Dark Oak Wood, Brown/Red Mushrooms, Apples

## Mobs

| Type     | Mobs                        |
|----------|-----------------------------|
| Passive  | Cow, Pig, Sheep                   |
| Neutral  | None                   |
| Hostile  | Zombie, Skeleton, Creeper, Spider, Witch, Phantom (night)                   |
