# Savanna

A warm, dry biome with acacia trees and tall grass. Never rains here. Villages are common, and llamas roam the plateau variants.

## Overview

| Field         | Value                       |
|---------------|-----------------------------|
| Dimension     | Overworld        |
| Category      | Savanna         |
| Temperature   | 2.0      |
| Precipitation | None (dry)               |
