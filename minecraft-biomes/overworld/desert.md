# Desert

Vast sandy biome with no passive animal spawns, cacti, and dead bushes. Villages, temples, and wells dot the landscape.

## Overview

| Field         | Value                       |
|---------------|-----------------------------|
| Dimension     | Overworld        |
| Category      | Dry         |
| Temperature   | 2.0      |
| Precipitation | None (dry)               |

## Blocks & Resources

**Surface / Terrain Blocks:** Sand, Sandstone, Hardened Sandstone

**Notable Resources:** Sand, Sandstone, Cactus, Dead Bush, Gold (from temples)
