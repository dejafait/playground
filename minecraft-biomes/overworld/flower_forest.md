# Flower Forest

A rarer forest variant bursting with every type of flower. A great source of dyes and a honey hotspot due to abundant bee nests.

## Overview

| Field         | Value                       |
|---------------|-----------------------------|
| Dimension     | Overworld        |
| Category      | Temperate         |
| Temperature   | 0.7      |
| Precipitation | Rain               |

## Blocks & Resources

**Surface / Terrain Blocks:** Grass Block, Dirt, Stone

**Notable Resources:** All Flower Types, Oak Wood, Birch Wood, Honeycomb
