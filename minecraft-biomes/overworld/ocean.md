# Ocean

The standard deep ocean biome with kelp forests, sea grass, and ocean monuments. Guardians and elder guardians patrol monuments.

## Overview

| Field         | Value                       |
|---------------|-----------------------------|
| Dimension     | Overworld        |
| Category      | Aquatic         |
| Temperature   | 0.5      |
| Precipitation | Rain               |

## Blocks & Resources

**Surface / Terrain Blocks:** Water, Sand, Gravel, Clay (deep)

**Notable Resources:** Kelp, Sea Grass, Prismarine (from monuments), Sponge, Cod

## Mobs

| Type     | Mobs                        |
|----------|-----------------------------|
| Passive  | Cod, Salmon, Squid, Dolphin                   |
| Neutral  | Dolphin                   |
| Hostile  | Drowned, Guardian (near monument)                   |

## Structures

Ocean Monument, Shipwreck, Buried Treasure
