# Deep Dark

The deepest and most dangerous cave biome, found below Y=0. Home to the Warden — the strongest hostile mob. Sculk blocks and Ancient Cities generate here.

## Overview

| Field         | Value                       |
|---------------|-----------------------------|
| Dimension     | Overworld        |
| Category      | Cave         |
| Temperature   | 0.8      |
| Precipitation | Rain               |

**Notable Resources:** Sculk (all variants), Echo Shard, Swift Sneak Enchantment (from chests), Reinforced Deepslate
