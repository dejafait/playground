# End Midlands

Medium-sized outer End islands with chorus plants. End Cities can spawn here, offering Shulker boxes and Elytra.

## Overview

| Field         | Value                       |
|---------------|-----------------------------|
| Dimension     | End        |
| Category      | End         |
| Temperature   | 0.5      |
| Precipitation | None (dry)               |

## Blocks & Resources

**Surface / Terrain Blocks:** End Stone

**Notable Resources:** Chorus Fruit, End Stone, Purpur Block, Shulker Shell, Elytra

## Mobs

| Type     | Mobs                        |
|----------|-----------------------------|
| Passive  | None                   |
| Neutral  | Enderman                   |
| Hostile  | Shulker                   |

## Structures

End City, End Ship (with Elytra)
