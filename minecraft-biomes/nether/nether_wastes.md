# Nether Wastes

The default Nether biome: a hellscape of netherrack, lava seas, and ghast-filled skies. Nether fortresses and quartz ore are the main draws.

## Overview

| Field         | Value                       |
|---------------|-----------------------------|
| Dimension     | Nether        |
| Category      | Nether         |
| Temperature   | 2.0      |
| Precipitation | None (dry)               |

## Blocks & Resources

**Surface / Terrain Blocks:** Netherrack, Gravel, Lava, Soul Sand (patches)

**Notable Resources:** Nether Quartz, Nether Gold Ore, Magma Block, Glowstone

## Mobs

| Type     | Mobs                        |
|----------|-----------------------------|
| Passive  | Strider (lava)                   |
| Neutral  | Zombie Piglin, Enderman                   |
| Hostile  | Ghast, Blaze (fortress), Wither Skeleton (fortress), Magma Cube                   |

## Structures

Nether Fortress, Ruined Portal, Bastion Remnant
