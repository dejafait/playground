# Playground

A collection of reference data and scripts.

## Contents

### `hello.py`
A hello world Python script.

### `colors/`
10 text files (one per color) containing the HTML hex code and multilingual translations for each color: black, blue, brown, green, orange, pink, purple, red, white, yellow.

### `minecraft-biomes/`
46 Markdown reference files describing Minecraft biomes, organized by dimension:

- `overworld/` — 36 biomes (plains, desert, jungle, ocean variants, caves, mountain peaks, etc.)
- `nether/` — 5 biomes (nether wastes, crimson forest, warped forest, basalt deltas, soul sand valley)
- `end/` — 5 biomes (the end, end barrens, end midlands, end highlands, small end islands)

Each file includes an overview table with dimension, category, temperature, and other properties, plus descriptions of terrain, structures, and mobs.

### `minecraft-mobs/`
80 text files, one per Minecraft mob, covering all passive, neutral, hostile, and boss mobs (from allay to zombified piglin). Each file includes mob type, category, health, and other stats.
